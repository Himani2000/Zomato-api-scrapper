## Zomato-api-scrapper

```

The scrapper extracts  the details of all the restaurants in a particular city. 
Aim is to fetch the data a user want and apply data analytics 
as per their data.



```


## Running the file

```
python main.py <api_key> <city_name> <max_Results>

```
